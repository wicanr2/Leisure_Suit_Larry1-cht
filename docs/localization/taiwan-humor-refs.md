# 台式笑話在地化參考素材庫

《幻想空間》(Leisure Suit Larry in the Land of the Lounge Lizards) 中文化的「風格參照 + 可改編梗庫」。主角賴瑞是穿廉價白西裝、把妹屢敗的中年魯蛇，原作靠性暗示、雙關與自嘲的黑色幽默撐場。本檔蒐集台灣本土幽默素材，讓後續重譯能把英文冷笑話換成台灣觀眾一看就懂、會心一笑的口語笑點。

這份是**梗庫與手法參考，不是最終譯文**。每條標註出處性質：

- **[考據]** — 已由網路來源查證的事實（諺語、流行語、原作台詞）。
- **[風格參照]** — 調性/句法特徵摘要，非逐字引用，年代久遠或無法精確溯源者歸此類。
- **[改編示範]** — 為本專案量身寫的套用句式範例，供重譯時參考改寫，不是既有笑話考據。

## 二、梗庫分類（可改編梗類型 + 可套用句式）

> 用法：左欄是手法/句式，右欄標來源性質。改編時先抓「手法」，再換成當前對白的具體對象（酒保、妞、賴瑞自己、死法）。

### 2.1 兩性把妹（搭訕、約會、被打槍）

1. **鋪陳→打槍反轉**（讀者文摘兩段式）：前句賴瑞自我感覺良好，後句被一句話戳破。
   - 例：「我跟她眼神交會了整整三秒——直到她發現我在看她後面那台提款機。」 [改編示範]
2. **時間差/曖昧雙關**：把「在一起」拆成字面與感情兩義。（源自台灣愛情笑話「分便當=在一起？」的手法） [風格參照]
   - 句式：把一個中性動作講得像告白，對方一臉問號。
3. **算命反諷把妹**：對方一句話反殺。（源自笑話「妳算什麼男人」的反向嘲諷） [風格參照]
   - 句式：賴瑞問「妳看我像不像好男人」→ 對方「你先像個男人再說」。
4. **現實主義幻滅**（自嘲式）：浪漫幻想被金錢/現實一秒擊碎。 [風格參照]
   - 句式：「原來她要的不是真愛——是我皮夾裡那張還沒過期的信用卡。」 [改編示範]
5. **撩妹金句的反用**：正經撩妹語錄拿來被打臉，製造反差。（台灣撩妹語錄文化） [風格參照]
6. **老派過氣感**：賴瑞用 80s 的自以為潮台詞，反而顯得 LKK。 [改編示範]
   - 句式：塞一句當年很潮、現在很俗的流行語（「這個妞…很ㄅ一ㄤ！」），突顯他跟不上時代。

### 2.2 酒吧買醉（點酒、喝掛、壯膽）

1. **貪杯諺語點題**：台語有「賭鬼假博繳、酒鬼假戒酒」一類「惡習難改」的反諷結構。 [考據/風格參照]
   - 句式：賴瑞發誓戒酒的下一格就又點一杯，旁白補一句「他戒酒的決心，跟他的髮線一樣，一年比一年後退。」 [改編示範]
2. **借酒壯膽→更慘**：喝完更敢，結果更糗的反差。 [改編示範]
   - 句式：「三杯黃湯下肚，賴瑞覺得自己是全場最帥的男人。可惜全場只有他一個男人。」
3. **廣告 jingle 洗腦腔**：仿當年感冒藥/提神飲料廣告的重複句，套在酒上。 [風格參照]
   - 句式：「悶的時候喝一杯，爽的時候喝一杯，把不到妹更要喝一杯～」（仿「用斯斯」重複句法）
4. **酒保吐槽客人**：酒保一句冷回。（讀者文摘〈各行各業〉職場吐槽調性） [風格參照]
   - 句式：賴瑞「給我來杯最烈的」→ 酒保「先生，我們這最烈的是你的體味。」 [改編示範]
5. **醉話雙關**：把醉後口齒不清做成諧音笑點。 [改編示範]

### 2.3 自嘲魯蛇（賴瑞的核心人設）

1. **沒人理他的自嘲**（對齊原作 "Nobody pays any attention to me... When did you get here?"）：講一句悲情又被無視。 [考據對齊]
   - 句式：「我這種男人，走在路上連野狗都懶得吠。」 [改編示範]
2. **婚姻/前女友怨念**（讀者文摘夫妻題材 + 原作 crude marriage joke）：拿失敗感情自嘲，點到為止。 [考據對齊/改編示範]
   - 句式：「我前女友說我像頭獎——很多人夢想中，但沒人真的中過。」
3. **宿命自嘲諺語**（台語）：用現成諺語一句到位。 [考據]
   - 「人衰，種瓠仔生菜瓜」（衰起來連種瓠瓜都長成菜瓜，什麼都不順）——賴瑞衰運旁白神句。
   - 「要哮無目屎，要哭無路來」（想哭都哭不出來的無奈）——把不到妹的悶。
   - 「三年一閏，好歹照輪」（好壞輪流轉）——賴瑞用來自我安慰，下一秒又踩雷。
4. **外貌自嘲**：白西裝、髮線、中年肚。 [改編示範]
   - 句式：「這件白西裝是我唯一的行頭，也是我唯一還願意跟我出門的東西。」
5. **圓仔花不知醜**（台語諺語）：「圓仔花不知醜，大紅花醜不知」（不自量力、沒自知之明）——旁白吐槽賴瑞自我感覺良好。 [考據]

### 2.4 死法吐槽（旁白冷嘲，LSL1 招牌）

> 原作旁白會用挖苦口氣播報賴瑞的各種蠢死法。台式對應：**報紙社會版標題腔 + 命理腔 + 綜藝誇張腔**。

1. **社會版標題腔**：仿 80s–90s 報紙社會新聞的「驚傳」「命危」「送醫不治」誇張標題。 [風格參照]
   - 句式：「驚傳！男子疑因把妹不成，慘遭自己的爛梗笑死，全場無人搶救。」 [改編示範]
2. **命理腔死法**：仿算命師語氣宣判。（台灣算命套話文化） [風格參照]
   - 句式：「我早就看你印堂發黑——果然，走不出這條巷子。」
3. **綜藝誇張旁白**：仿《我猜》主持人語氣，「人不可貌相，海水不可斗量」式套句反用。 [考據語感]
   - 句式：「人不可貌相——尤其是躺在地上這位，現在真的很難相了。」 [改編示範]
4. **廣告收尾腔**：死得莫名，配一句廣告 tagline 反諷。 [改編示範]
   - 句式：「這，就是不聽勸的下場。（本則旁白由賴瑞的智商贊助播出。）」
5. **黑色職業玩笑**（對齊原作律師/鯊魚「專業禮遇」冷笑話）：把死法連到某種黑色諷刺。 [考據對齊]

### 2.5 雙關諧音（黃色笑話的主力手法）

> 賴瑞的性暗示原作就是靠 double entendre。台式做法：**用激骨話「謎面—謎底」結構，靠諧音轉義，不直說。**

1. **激骨話結構直接借用**（台語，全部 [考據]）：
   - 「二十兩」→「斤四」（諧音「近視」）：可玩「看不清」的雙關。
   - 「十二月天睏厝頂」→「凍霜」（雙關：既是冷，也罵「小氣、吝嗇」）：吐槽賴瑞捨不得花錢。
   - 「乞食揹葫蘆」→「假仙」（裝模作樣）：吐槽賴瑞裝紳士。
   - 「大某拍細姨」→「大出手」（雙關：出手大方）：可轉用在「出手闊綽 vs 動手」的曖昧。
   - 「大兄無坐正」→「歪哥」（諧音，指貪不正之利）：吐槽賴瑞心術不正。
2. **國語同音字借位**：同音不同義製造想歪空間，但字面乾淨。（台灣諧音文化） [考據手法]
   - 手法：講一個字面正經、聽起來會歪的詞，讓玩家自己接。「點到為止」全靠這一步。
3. **多語諧音**（國/台/英夾雜）：台灣特色，跨語言諧音。 [考據手法]
   - 例：英文詞用台語諧音重新詮釋（如原作 "briefs" 律師/內褲雙關，可找台語對應的「褲底」類諧音改寫）。
4. **拆字諧音**：把一個詞拆開重組出黃色聯想（源自笑話「大伯起/大勃起」的拆字手法）。 [風格參照]
   - 尺度提醒：這類最容易過火，改編時把露骨那半留白，只給諧音提示。

### 2.6 台語俚語（提味用，少量點綴）

> 原則：**整句國語為主，關鍵情緒點插一個台語詞**，不要整段台語（玩家未必都懂）。以下皆 [考據] 詞義。

1. **歹勢**（pháinn-sè）：不好意思／害臊。賴瑞搭訕碰壁時的口頭禪。
2. **𨑨迌**（tshit-thô）：玩耍／鬼混，也可指不務正業。賴瑞「來 Lost Wages 𨑨迌」。
3. **切心**（tsheh-sim）：心寒、灰心。被打槍後「賴瑞真正切心」。
4. **凍未條**（tòng bē tiâu）：受不了、忍不住。看到辣妹「賴瑞凍未條」。
5. **語尾助詞調口氣**：「齁」（敷衍/撒嬌）、「溜」（俏皮的「了唷」連音）、「啦」——加在句尾讓對白更口語、更台。 [考據]
   - 例：「好啦好啦，我知道我沒救了啦～」比「我知道我沒救了」更有戲。
6. **衰運/面子諺語**（前面 2.3 已列，可穿插）：「乞食趕廟公」（喧賓奪主、不自量力）等，適合旁白吐槽賴瑞。 [考據]

## 三、尺度建議（哪些能用、哪些要收）

**適合放進遊戲對白：**

- **雙關與諧音的性暗示**：字面乾淨、靠聯想，是原作與台灣民間笑話共通的「點到為止」正解。優先用這類。
- **自嘲魯蛇**：賴瑞越慘越好笑，這條幾乎不會過火，可以大量用。
- **夫妻/約會/酒吧的生活反差笑點**（讀者文摘調性）：溫和、普世、好翻，安全牌。
- **旁白挖苦玩家/死法吐槽**：保住原作靈魂，用報紙腔/命理腔/綜藝腔在地化，效果好又不低俗。
- **少量台語詞提味**：情緒點插一個，加分。

**要收斂或改寫：**

- **直白露骨的器官/動作描述**：原作靠暗示，直說反而不好笑也失格調。拆字諧音（如「大勃起」型）把露骨那半留白。
- **整段台語**：玩家理解門檻高，會打斷節奏。台語只當調味，不當主體。
- **過時到看不懂的流行語**：80s–90s 梗要挑「現在還聽得懂」的（LKK、遜斃了 OK；太冷僻的綜藝梗會 miss）。
- **涉及真實人物/政治/族群的諧音**：容易引戰或冒犯，避開。賴瑞的葷素笑點對準「賴瑞自己很衰很色」即可，別掃到真人。
- **歧視性玩笑**：原作有些年代性的冒犯玩笑（如身材、族群），在地化時淡化或改成賴瑞自身的糗，不要原樣複製。

**一句話原則**：色在雙關、笑在自嘲、賤在旁白——露骨留白，台語提味，年代感點到為止。
